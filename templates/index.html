Explainable AI for Chest X-Ray Diagnosis

Cardiomegaly Detection with Trust Evaluation

About This System

This system uses a Convolutional Neural Network (CNN) to detect Cardiomegaly (enlarged heart) from chest X-ray images. To improve trust in AI decisions, multiple Explainable AI (XAI) methods such as Saliency Maps, Grad-CAM, LIME, and SHAP are used.

The system also evaluates whether the model focuses on clinically relevant regions of the image and generates a natural language explanation using a large language model.
